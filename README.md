# 3D Website

A modern and interactive 3D website built with React, TypeScript, and Vite. This project features stunning 3D elements and smooth animations for an engaging user experience.

## Live Demo

Check out the deployed version here:  
[https://3-d-website-i9qg.vercel.app/]

## Features

- Fast development with Vite and Hot Module Replacement (HMR)
- Type-safe code using TypeScript
- Linting with ESLint for clean and consistent code
- Responsive design with interactive 3D visuals

## Getting Started

### Prerequisites

- Node.js (v18 or later recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AhmadReza16/3D-website.git
   cd 3D-website
   ```

1:Install dependencies:
Bash
npm install
2:Start the development
Bash
npm run dev

Open http://localhost:5173 to view it in the browser.

Build for Production

Bash
npm run build

The build output will be in the dist folder.

Preview Production Build

Bash
npm run preview
